🎓 College Appointment System API

An efficient backend API to manage student-professor appointments seamlessly.

📄 Swagger Docs 💻 GitHub Repo

🚀 Project Overview

The College Appointment System API simplifies the process of managing appointments between students and professors. Built with modern technologies, it ensures smooth authentication, appointment booking, slot management, and status tracking.

🔧 Tech Stack

  • ⚡ Node.js + Express
  • 🗄️ MongoDB (Database)
  • 🔐 JWT Authentication
  • 🧪 Jest + Postman Automated Tests
  • 🌐 Swagger API Documentation

📌 Features

  • ✔️ Student & Professor Authentication
  • ✔️ Appointment Booking & Cancellation
  • ✔️ Availability Slot Management
  • ✔️ Appointment Status Tracking
  • ✔️ Automated API Tests
